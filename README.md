# Sliding-Puzzle-Game-Java
Console-based 15 Puzzle game in Java where players arrange numbered tiles in correct order using logic and move validation.
# 🧩 Sliding Puzzle (15 Puzzle) - Java Console

## 📌 Description
A classic 15 Puzzle game developed using Core Java.  
The game consists of a 4x4 grid with numbered tiles and one empty space. The player must arrange the tiles in order.

## 🎮 Features
- 4x4 grid puzzle
- Tile movement (up, down, left, right)
- Move validation
- Winning condition check

## 🛠 Concepts Used
✔ 2D Array  
✔ Logic Building  
✔ Conditional Statements  
✔ Loops  
✔ User Input  

## 🎯 Game Objective
Arrange the numbers from 1 to 15 in correct order.

## ▶ How to Run
1. Open project in Eclipse / IntelliJ
2. Run the main class
